## Advanced Mode - Basic

Basic advanced mode with system and user messages.

    {

## Advanced Mode - Multiple Messages

Advanced mode with multiple message exchanges.

<LLMPlayground 
  defaultMode="advanced"
  defaultMessages={[
    {
      role: "system",
      content: "You are a coding tutor."
    },
    {
      role: "user",
      content: "What is a closure in JavaScript?"
    },
    {
      role: "assistant",
      content: "A closure is a function that has access to variables in its outer (enclosing) lexical scope, even after the outer function has returned."
    },
    {
      role: "user",
      content: "Can you give me an example?"
    }
  ]}
  height="600px"
/>
